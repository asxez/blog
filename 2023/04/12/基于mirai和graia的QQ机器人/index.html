# 功能

1.ChatGPT API 版（无上下文）（需魔法） /gpt
2.New Bing （有上下文）（需魔法） /bing
3.AI 绘画 /aip
4. 随机动漫图片
5. 基本翻译 /tra
6. 人工图库（我亲手挑选的）
7. 音乐（目前支持酷我和网易云）/kw /wyy
8. 随机一言
9. 群违规词检测
10. 入群申请检测与审核 （我设的是 QQ 等级大于三颗星的才能通过，你们可以自行修改）
11. 定时推送实况天气
12. 定时发送笑话段子（时间设在 14：20，因为大家才睡了午觉没啥精神）
13./kick /ma /uma /mo /umo /tra /shutdown
14. 控制端后门
更多功能正在开发中，欢迎大家评论区提建议哦
………………

# 部分功能使用方法及解释

ChatGPT：/gpt 问题
New Bing：/bing 问题
ai 绘画：/aip 描述
音乐：/kw 音乐名 和 /wyy 音乐名
翻译：/tra en 文本 （将文本翻译为英文） 和 /tra zh 文本 （将文本翻译为中文）
关机：/shutdown （仅系统管理员可用）
清除 new bing 记忆或者重启程序：/clean
控制端后门：/s 文本 这段命令可以让机器人将文本发送至指定群聊，仅系统管理员可用（这段指令只能私发给机器人）
/kick @xxx 踢出某人（注意指令和艾特间的空格）
/ma 开启全体禁言
/uma 关闭全体禁言
/mo @xxx 时间 （禁言某成员，时间单位为分钟）
/umo @xxx 解除某成员禁言
