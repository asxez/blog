# 功能

  1. ChatGPT API 版（无上下文）（需魔法） /gpt
  2. New Bing （有上下文）（需魔法） /bing
  3. 音乐（目前支持酷我和网易云）/kw /wyy
  4. 群违规词检测
  5. 入群申请检测与审核
  6. /kick /ma /uma /mo /umo /tra /shutdown
  7. 控制端后门

# 部分功能使用方法及解释

ChatGPT：/gpt 问题
New Bing：/bing 问题
音乐：/kw 音乐名 和 /wyy 音乐名
关机：/shutdown （仅系统管理员可用）
清除 new bing 记忆或者重启程序：/clean
控制端后门：/s 文本 这段命令可以让机器人将文本发送至指定群聊，仅系统管理员可用（这段指令只能私发给机器人）
/kick @xxx 踢出某人（注意指令和艾特间的空格）
/ma 开启全体禁言
/uma 关闭全体禁言
/mo @xxx 时间 （禁言某成员，时间单位为分钟）
/umo @xxx 解除某成员禁言
